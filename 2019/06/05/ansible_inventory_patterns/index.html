Host Inventory

Host Inventory й…ҚзҪ®ж–Үд»¶пјҡй»ҳи®Өж–Үд»¶/etc/ansible/hostsпјҢз”ЁжқҘе‘ҠиҜүAnsibleйңҖиҰҒз®ЎзҗҶе“Әдәӣдё»жңәгҖӮе№¶дё”жҠҠиҝҷдәӣдё»жңәж №жҚ®жҢүйңҖеҲҶзұ»гҖӮ

еҸҜд»Ҙж №жҚ®з”ЁйҖ”еҲҶзұ»пјҡж•°жҚ®еә“иҠӮзӮ№пјҢжңҚеҠЎиҠӮзӮ№зӯүпјӣж №жҚ®ең°зӮ№еҲҶзұ»пјҡдёӯйғЁпјҢиҘҝйғЁжңәжҲҝгҖӮ

AnsibleеҸҜд»ҘеҗҢж—¶ж“ҚдҪңеұһдәҺдёҖдёӘз»„зҡ„еӨҡеҸ°дё»жңәпјҢз»„е’Ңдё»жңәд№Ӣй—ҙзҡ„е…ізі»йҖҡиҝҮ inventory ж–Үд»¶й…ҚзҪ®гҖӮ

иҝңзЁӢдё»жңәе’Ңз»„

з®ҖеҚ•еҲҶз»„пјҡ
ж–№жӢ¬еҸ·[]дёӯжҳҜз»„еҗҚпјҢз”ЁдәҺеҜ№з»„иҝӣиЎҢзі»з»ҹеҲҶзұ»пјҢдҫҝдәҺеҜ№дёҚеҗҢзі»з»ҹиҝӣиЎҢеҲҶејҖзҡ„з®ЎзҗҶ

дёҖз»„зӣёдјјзҡ„ hostnameз®ҖеҶҷж–№жі•пјҡ

SSHз«ҜеҸЈдёҚжҳҜ22зҡ„иЎЁзӨәж–№жі•пјҡ

еҚ•дёӘдё»жңәи®ҫзҪ®еҲ«еҗҚж–№ејҸпјҡ

еңЁиҝҷдёӘдҫӢеӯҗдёӯпјҢйҖҡиҝҮ вҖңjumperвҖқ еҲ«еҗҚпјҢдјҡиҝһжҺҘ 192.168.1.50:5555
пјҲиҝҷжҳҜйҖҡиҝҮ inventory ж–Үд»¶зҡ„зү№жҖ§еҠҹиғҪи®ҫзҪ®зҡ„еҸҳйҮҸгҖӮдёҖиҲ¬иҖҢиЁҖ,иҝҷдёҚжҳҜи®ҫзҪ®еҸҳйҮҸ(жҸҸиҝ°дҪ зҡ„зі»з»ҹзӯ–з•Ҙзҡ„еҸҳйҮҸ)зҡ„жңҖеҘҪж–№ејҸпјү

иҝңзЁӢдё»жңәзҡ„иҝһжҺҘеҸӮж•°е’ҢеҸҳйҮҸ

еҸӮж•°
жҢҮе®ҡServerзҡ„иҝһжҺҘеҸӮж•°пјҢе…¶дёӯеҢ…жӢ¬иҝһжҺҘж–№жі•пјҢз”ЁжҲ·зӯүгҖӮ

еҸҜд»ҘжҢҮе®ҡзҡ„еҸӮж•°еҸӮиҖғпјҡdocs.ansible.com

дёәдёҖдёӘз»„жҢҮе®ҡеҸҳйҮҸпјҡ

жҠҠдёҖдёӘз»„дҪңдёәеҸҰдёҖдёӘз»„зҡ„еӯҗжҲҗе‘ҳ

еҸҜд»ҘжҠҠдёҖдёӘз»„дҪңдёәеҸҰдёҖдёӘз»„зҡ„еӯҗжҲҗе‘ҳ,д»ҘеҸҠеҲҶй…ҚеҸҳйҮҸз»ҷж•ҙдёӘз»„дҪҝз”ЁпјҢдҫӢпјҡ
еҲҶз»„usaзҡ„еӯҗз»„иҝҳеҸҜд»ҘжҳҜе…¶е®ғзҡ„з»„пјҢдҫӢеҰӮ[usa:children]дёӯиҝҳеҸҜд»ҘеҢ…еҗ«еӯҗз»„southeast, [southeast:children]дёӯиҝҳеҸҜд»ҘеҢ…еҗ«atlantaе’Ңreleigh

жҢүзӣ®еҪ•з»“жһ„ з»„з»ҮHostе’ҢGroupеҸҳйҮҸ(more)

еңЁ inventory дё»ж–Үд»¶дёӯдҝқеӯҳжүҖжңүзҡ„еҸҳйҮҸе№¶дёҚжҳҜжңҖдҪізҡ„ж–№ејҸ.иҝҳеҸҜд»ҘдҝқеӯҳеңЁзӢ¬з«Ӣзҡ„ж–Үд»¶дёӯ,иҝҷдәӣзӢ¬з«Ӣж–Үд»¶дёҺ inventory ж–Үд»¶дҝқжҢҒе…іиҒ”. дёҚеҗҢдәҺ inventory ж–Үд»¶(INI ж јејҸ),иҝҷдәӣзӢ¬з«Ӣж–Үд»¶зҡ„ж јејҸдёә YAML.иҜҰи§Ғ YAML иҜӯжі• .

еҒҮи®ҫ inventory ж–Үд»¶зҡ„и·Ҝеҫ„дёә:

еҒҮи®ҫжңүдёҖдёӘдё»жңәеҗҚдёә вҖҳfoosballвҖҷ, дё»жңәеҗҢж—¶еұһдәҺдёӨдёӘз»„пјҢз»„зҡ„еҗҚеӯ—еҲҶеҲ«жҳҜ вҖҳraleighвҖҷе’Ң вҖҳwebserversвҖҷ.
йӮЈд№Ҳд»ҘдёӢз»„еҸҳйҮҸзҡ„й…ҚзҪ®ж–Үд»¶дёӯзҡ„еҸҳйҮҸеҸҜд»Ҙиў« вҖҳfoosballвҖҷ дё»жңәжүҖз”Ё.

дёӢйқўдҫқж¬Ўдёә вҖҳraleighвҖҷ зҡ„з»„еҸҳйҮҸ,вҖҷwebserversвҖҷ зҡ„з»„еҸҳйҮҸ,вҖҷfoosballвҖҷ зҡ„дё»жңәеҸҳйҮҸ зҡ„й…ҚзҪ®ж–Үд»¶:

дёҫдҫӢжқҘиҜҙ,еҒҮи®ҫдҪ жңүдёҖдәӣдё»жңә,еұһдәҺдёҚеҗҢзҡ„ж•°жҚ®дёӯеҝғ,е№¶дҫқж¬ЎиҝӣиЎҢеҲ’еҲҶ.жҜҸдёҖдёӘж•°жҚ®дёӯеҝғдҪҝз”ЁдёҖдәӣдёҚеҗҢзҡ„жңҚеҠЎеҷЁ.жҜ”еҰӮ ntp жңҚеҠЎеҷЁ, database жңҚеҠЎеҷЁзӯүзӯү. йӮЈд№Ҳ вҖҳraleighвҖҷ иҝҷдёӘз»„зҡ„з»„еҸҳйҮҸе®ҡд№үеңЁж–Үд»¶ вҖҳ/etc/ansible/group_vars/raleighвҖҷ д№Ӣдёӯ,еҸҜиғҪзұ»дјјиҝҷж ·:

иҝҷдәӣе®ҡд№үеҸҳйҮҸзҡ„ж–Үд»¶дёҚжҳҜдёҖе®ҡиҰҒеӯҳеңЁ,еӣ дёәиҝҷжҳҜеҸҜйҖүзҡ„зү№жҖ§.

иҝҳжңүжӣҙиҝӣдёҖжӯҘзҡ„иҝҗз”Ё,дҪ еҸҜд»ҘдёәдёҖдёӘдё»жңә,жҲ–дёҖдёӘз»„,еҲӣе»әдёҖдёӘзӣ®еҪ•,зӣ®еҪ•еҗҚе°ұжҳҜдё»жңәеҗҚжҲ–з»„еҗҚ.зӣ®еҪ•дёӯзҡ„еҸҜд»ҘеҲӣе»әеӨҡдёӘж–Үд»¶, ж–Үд»¶дёӯзҡ„еҸҳйҮҸйғҪдјҡиў«иҜ»еҸ–дёәдё»жңәжҲ–з»„зҡ„еҸҳйҮҸ.еҰӮдёӢ вҖҳraleighвҖҷ з»„еҜ№еә”дәҺ /etc/ansible/group_vars/raleigh/ зӣ®еҪ•,е…¶дёӢжңүдёӨдёӘж–Үд»¶ db_settings е’Ң cluster_settings, е…¶дёӯеҲҶеҲ«и®ҫзҪ®дёҚеҗҢзҡ„еҸҳйҮҸ:

вҖҳraleighвҖҷ з»„дёӢзҡ„жүҖжңүдё»жңә,йғҪеҸҜд»ҘдҪҝз”Ё вҖҳraleighвҖҷ з»„зҡ„еҸҳйҮҸ.еҪ“еҸҳйҮҸеҸҳеҫ—еӨӘеӨҡж—¶,еҲҶж–Үд»¶е®ҡд№үеҸҳйҮҸжӣҙж–№дҫҝжҲ‘д»¬иҝӣиЎҢз®ЎзҗҶе’Ңз»„з»Ү. иҝҳжңүдёҖдёӘж–№ејҸд№ҹеҸҜеҸӮиҖғ,иҜҰи§Ғ Ansible Vault е…ідәҺз»„еҸҳйҮҸзҡ„йғЁеҲҶ. жіЁж„Ҹ,еҲҶж–Үд»¶е®ҡд№үеҸҳйҮҸзҡ„ж–№ејҸеҸӘйҖӮз”ЁдәҺ Ansible 1.4 еҸҠд»ҘдёҠзүҲжң¬.

Tip: Ansible 1.2 еҸҠд»ҘдёҠзҡ„зүҲжң¬дёӯ,group_vars/ е’Ң host_vars/ зӣ®еҪ•еҸҜж”ҫеңЁ inventory зӣ®еҪ•дёӢ,жҲ–жҳҜ playbook зӣ®еҪ•дёӢ. еҰӮжһңдёӨдёӘзӣ®еҪ•дёӢйғҪеӯҳеңЁ,йӮЈд№Ҳ playbook зӣ®еҪ•дёӢзҡ„й…ҚзҪ®дјҡиҰҶзӣ– inventory зӣ®еҪ•зҡ„й…ҚзҪ®.

Tip: жҠҠдҪ зҡ„ inventory ж–Үд»¶ е’Ң еҸҳйҮҸ ж”ҫе…Ҙ git repo дёӯ,д»Ҙдҫҝи·ҹиёӘд»–д»¬зҡ„жӣҙж–°,иҝҷжҳҜдёҖз§ҚйқһеёёжҺЁиҚҗзҡ„ж–№ејҸ.
