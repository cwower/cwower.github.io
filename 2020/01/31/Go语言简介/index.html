з®Җд»Ӣ

GoogleејҖеҸ‘зҡ„ејҖжәҗзј–зЁӢиҜӯиЁҖ
иҜһз”ҹдәҺ2006е№ҙ1жңҲ2ж—ҘдёӢеҚҲ15зӮ№4еҲҶ5з§’пјҢдәҺ2009е№ҙ11жңҲејҖжәҗпјҢ2012е№ҙеҸ‘еёғgoзЁіе®ҡзүҲгҖӮGoиҜӯиЁҖеңЁеӨҡж ёе№¶еҸ‘дёҠжӢҘжңүеҺҹз”ҹзҡ„и®ҫи®ЎдјҳеҠҝпјҢGoиҜӯиЁҖд»Һеә•еұӮеҺҹз”ҹж”ҜжҢҒе№¶еҸ‘пјҢж— йЎ»з¬¬дёүж–№еә“гҖҒејҖеҸ‘иҖ…зҡ„зј–зЁӢжҠҖе·§е’ҢејҖеҸ‘з»ҸйӘҢгҖӮ
е®ғзҡ„дё»иҰҒзӣ®ж ҮжҳҜвҖңе…је…·Python зӯүеҠЁжҖҒиҜӯиЁҖзҡ„ејҖеҸ‘йҖҹеәҰе’ҢC/C++зӯүзј–иҜ‘еһӢиҜӯиЁҖзҡ„жҖ§иғҪдёҺе®үе…ЁжҖ§вҖқ

GoиҜӯиЁҖдёәе№¶еҸ‘иҖҢз”ҹ вҖ”вҖ”21дё–зәӘзҡ„CиҜӯиЁҖ
зЎ¬д»¶еҲ¶йҖ е•ҶжӯЈеңЁдёәеӨ„зҗҶеҷЁж·»еҠ и¶ҠжқҘи¶ҠеӨҡзҡ„еҶ…ж ёд»ҘжҸҗй«ҳжҖ§иғҪгҖӮжүҖжңүж•°жҚ®дёӯеҝғйғҪеңЁиҝҷдәӣеӨ„зҗҶеҷЁдёҠиҝҗиЎҢпјҢжӣҙйҮҚиҰҒзҡ„жҳҜпјҢд»ҠеӨ©зҡ„еә”з”ЁзЁӢеәҸдҪҝз”ЁеӨҡдёӘеҫ®жңҚеҠЎжқҘз»ҙжҠӨж•°жҚ®еә“иҝһжҺҘпјҢж¶ҲжҒҜйҳҹеҲ—е’Ңз»ҙжҠӨзј“еӯҳгҖӮеӣ жӯӨпјҢејҖеҸ‘зҡ„иҪҜд»¶е’Ңзј–зЁӢиҜӯиЁҖеә”иҜҘеҸҜд»ҘиҪ»жқҫең°ж”ҜжҢҒе№¶еҸ‘жҖ§пјҢе№¶дё”еә”иҜҘиғҪеӨҹйҡҸзқҖCPUж ёеҝғж•°йҮҸзҡ„еўһеҠ иҖҢеҸҜжү©еұ•гҖӮ

дҪҶжҳҜпјҢеӨ§еӨҡж•°зҺ°д»Јзј–зЁӢиҜӯиЁҖпјҲеҰӮJavaпјҢPythonзӯүпјүйғҪжқҘиҮӘ90е№ҙд»Јзҡ„еҚ•зәҝзЁӢзҺҜеўғгҖӮиҷҪз„¶дёҖдәӣзј–зЁӢиҜӯиЁҖзҡ„жЎҶжһ¶еңЁдёҚж–ӯең°жҸҗй«ҳеӨҡж ёиө„жәҗдҪҝз”Ёж•ҲзҺҮпјҢдҫӢеҰӮ Java зҡ„ Netty зӯүпјҢдҪҶд»Қз„¶йңҖиҰҒејҖеҸ‘дәәе‘ҳиҠұиҙ№еӨ§йҮҸзҡ„ж—¶й—ҙе’ҢзІҫеҠӣжҗһжҮӮиҝҷдәӣжЎҶжһ¶зҡ„иҝҗиЎҢеҺҹзҗҶеҗҺжүҚиғҪзҶҹз»ғжҺҢжҸЎгҖӮ

GoдәҺ2009е№ҙеҸ‘еёғпјҢеҪ“ж—¶еӨҡж ёеӨ„зҗҶеҷЁе·Із»ҸдёҠеёӮгҖӮGoиҜӯиЁҖеңЁеӨҡж ёе№¶еҸ‘дёҠжӢҘжңүеҺҹз”ҹзҡ„и®ҫи®ЎдјҳеҠҝпјҢGoиҜӯиЁҖд»Һеә•еұӮеҺҹз”ҹж”ҜжҢҒе№¶еҸ‘пјҢж— йЎ»з¬¬дёүж–№еә“гҖҒејҖеҸ‘иҖ…зҡ„зј–зЁӢжҠҖе·§е’ҢејҖеҸ‘з»ҸйӘҢгҖӮ

еҫҲеӨҡе…¬еҸёпјҢзү№еҲ«жҳҜдёӯеӣҪзҡ„дә’иҒ”зҪ‘е…¬еҸёпјҢеҚіе°ҶжҲ–иҖ…е·Із»Ҹе®ҢжҲҗдәҶдҪҝз”Ё Go иҜӯиЁҖж”№йҖ ж—§зі»з»ҹзҡ„иҝҮзЁӢгҖӮз»ҸиҝҮ Go иҜӯиЁҖйҮҚжһ„зҡ„зі»з»ҹиғҪдҪҝз”Ёжӣҙе°‘зҡ„зЎ¬д»¶иө„жәҗиҺ·еҫ—жӣҙй«ҳзҡ„е№¶еҸ‘е’ҢI/OеҗһеҗҗиЎЁзҺ°гҖӮе……еҲҶжҢ–жҺҳзЎ¬д»¶и®ҫеӨҮзҡ„жҪңеҠӣд№ҹж»Ўи¶іеҪ“еүҚзІҫз»ҶеҢ–иҝҗиҗҘзҡ„еёӮеңәеӨ§зҺҜеўғгҖӮ

GoиҜӯиЁҖзҡ„е№¶еҸ‘жҳҜеҹәдәҺ goroutine зҡ„пјҢgoroutine зұ»дјјдәҺзәҝзЁӢпјҢдҪҶе№¶йқһзәҝзЁӢгҖӮеҸҜд»Ҙе°Ҷ goroutine зҗҶи§ЈдёәдёҖз§ҚиҷҡжӢҹзәҝзЁӢгҖӮGo иҜӯиЁҖиҝҗиЎҢж—¶дјҡеҸӮдёҺи°ғеәҰ goroutineпјҢе№¶е°Ҷ goroutine еҗҲзҗҶең°еҲҶй…ҚеҲ°жҜҸдёӘ CPU дёӯпјҢжңҖеӨ§йҷҗеәҰең°дҪҝз”ЁCPUжҖ§иғҪгҖӮејҖеҗҜдёҖдёӘgoroutineзҡ„ж¶ҲиҖ—йқһеёёе°ҸпјҲеӨ§зәҰ2KBзҡ„еҶ…еӯҳпјүпјҢдҪ еҸҜд»ҘиҪ»жқҫеҲӣе»әж•°зҷҫдёҮдёӘgoroutineгҖӮ

goroutineзҡ„зү№зӮ№пјҡ

  1. goroutineе…·жңүеҸҜеўһй•ҝзҡ„еҲҶж®өе Ҷж ҲгҖӮиҝҷж„Ҹе‘ізқҖе®ғд»¬еҸӘеңЁйңҖиҰҒж—¶жүҚдјҡдҪҝз”ЁжӣҙеӨҡеҶ…еӯҳгҖӮ
  2. goroutineзҡ„еҗҜеҠЁж—¶й—ҙжҜ”зәҝзЁӢеҝ«гҖӮ
  3. goroutineеҺҹз”ҹж”ҜжҢҒеҲ©з”Ёchannelе®үе…Ёең°иҝӣиЎҢйҖҡдҝЎгҖӮ
  4. goroutineе…ұдә«ж•°жҚ®з»“жһ„ж—¶ж— йңҖдҪҝз”Ёдә’ж–Ҙй”ҒгҖӮ

зј–иҜ‘еһӢиҜӯиЁҖ
дёҺе…¶д»–зҺ°д»Јй«ҳзә§иҜӯиЁҖпјҲеҰӮJava/PythonпјүзӣёжҜ”пјҢдҪҝз”ЁCпјҢC++зҡ„жңҖеӨ§еҘҪеӨ„жҳҜе®ғд»¬зҡ„жҖ§иғҪгҖӮеӣ дёәC/ C++жҳҜзј–иҜ‘еһӢиҜӯиЁҖиҖҢдёҚжҳҜи§ЈйҮҠзҡ„иҜӯиЁҖгҖӮ еӨ„зҗҶеҷЁеҸӘиғҪзҗҶи§ЈдәҢиҝӣеҲ¶ж–Үд»¶пјҢJavaе’ҢPythonиҝҷз§Қй«ҳзә§иҜӯиЁҖеңЁиҝҗиЎҢзҡ„ж—¶еҖҷйңҖиҰҒе…Ҳе°Ҷдәәзұ»еҸҜиҜ»зҡ„д»Јз Ғзҝ»иҜ‘жҲҗеӯ—иҠӮз ҒпјҢз„¶еҗҺз”ұдё“й—Ёзҡ„и§ЈйҮҠеҷЁеҶҚиҪ¬еҸҳжҲҗеӨ„зҗҶеҷЁеҸҜд»ҘзҗҶи§Јзҡ„дәҢиҝӣеҲ¶ж–Үд»¶гҖӮ

еҗҢC,C++дёҖж ·пјҢGoиҜӯиЁҖд№ҹжҳҜзј–иҜ‘еһӢзҡ„иҜӯиЁҖпјҢе®ғзӣҙжҺҘе°Ҷдәәзұ»еҸҜиҜ»зҡ„д»Јз Ғзј–иҜ‘жҲҗдәҶеӨ„зҗҶеҷЁеҸҜд»ҘзӣҙжҺҘиҝҗиЎҢзҡ„дәҢиҝӣеҲ¶ж–Үд»¶пјҢжү§иЎҢж•ҲзҺҮжӣҙй«ҳпјҢжҖ§иғҪжӣҙеҘҪгҖӮ

зү№зӮ№

иҜӯжі•з®ҖжҙҒ
Go иҜӯиЁҖз®ҖеҚ•жҳ“еӯҰпјҢеӯҰд№ жӣІзәҝе№ізј“пјҢдёҚйңҖиҰҒеғҸ C/C++ иҜӯиЁҖеҠЁиҫ„йңҖиҰҒдёӨеҲ°дёүе№ҙзҡ„еӯҰд№ жңҹгҖӮGo иҜӯиЁҖиў«з§°дёәвҖңдә’иҒ”зҪ‘ж—¶д»Јзҡ„CиҜӯиЁҖвҖқгҖӮGo иҜӯиЁҖзҡ„йЈҺж јзұ»дјјдәҺCиҜӯиЁҖгҖӮе…¶иҜӯжі•еңЁCиҜӯиЁҖзҡ„еҹәзЎҖдёҠиҝӣиЎҢдәҶеӨ§е№…зҡ„з®ҖеҢ–пјҢеҺ»жҺүдәҶдёҚйңҖиҰҒзҡ„иЎЁиҫҫејҸжӢ¬еҸ·пјҢеҫӘзҺҜд№ҹеҸӘжңү for дёҖз§ҚиЎЁзӨәж–№жі•пјҢе°ұеҸҜд»Ҙе®һзҺ°ж•°еҖјгҖҒй”®еҖјзӯүеҗ„з§ҚйҒҚеҺҶгҖӮ

д»Јз ҒйЈҺж јз»ҹдёҖ
Go иҜӯиЁҖжҸҗдҫӣдәҶдёҖеҘ—ж јејҸеҢ–е·Ҙе…·вҖ”вҖ”go fmtгҖӮдёҖдәӣ Go иҜӯиЁҖзҡ„ејҖеҸ‘зҺҜеўғжҲ–иҖ…зј–иҫ‘еҷЁеңЁдҝқеӯҳж—¶пјҢйғҪдјҡдҪҝз”Ёж јејҸеҢ–е·Ҙе…·иҝӣиЎҢдҝ®ж”№д»Јз Ғзҡ„ж јејҸеҢ–пјҢиҝҷж ·е°ұдҝқиҜҒдәҶдёҚеҗҢејҖеҸ‘иҖ…жҸҗдәӨзҡ„д»Јз ҒйғҪжҳҜз»ҹдёҖзҡ„ж јејҸгҖӮ(еҗҗж§ҪдёӢпјҡеҶҚд№ҹдёҚз”ЁжӢ…еҝғйӮЈдәӣзңӢдёҚжҮӮзҡ„й»‘йӯ”жі•дәҶвҖҰ)

ејҖеҸ‘ж•ҲзҺҮй«ҳ
GoиҜӯиЁҖе®һзҺ°дәҶејҖеҸ‘ж•ҲзҺҮдёҺжү§иЎҢж•ҲзҺҮзҡ„е®ҢзҫҺз»“еҗҲпјҢи®©дҪ еғҸеҶҷPythonд»Јз ҒпјҲж•ҲзҺҮпјүдёҖж ·зј–еҶҷCд»Јз ҒпјҲжҖ§иғҪпјүгҖӮ

Goзҡ„дҪҝз”ЁеңәжҷҜ

зӣ®еүҚGoиҜӯиЁҖе·Із»Ҹвјҙжіӣеә”з”ЁдәҺдәәе·ҘжҷәиғҪгҖҒдә‘и®Ўз®—ејҖеҸ‘гҖҒе®№еҷЁиҷҡжӢҹеҢ–гҖҒвјӨж•°жҚ®ејҖеҸ‘гҖҒж•°жҚ®еҲҶжһҗеҸҠз§‘еӯҰи®Ўз®—гҖҒиҝҗз»ҙејҖеҸ‘гҖҒзҲ¬иҷ«ејҖеҸ‘гҖҒжёёжҲҸејҖеҸ‘зӯүйўҶеҹҹгҖӮ
